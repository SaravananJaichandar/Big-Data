**Commands to Perform Map-Reduce** :

1.To run the jar file - hadoop jar "filename.jar" "packagename.classname" "Input file path in hdfs" "Output file path in hdfs"
2.Sorting the results - sort -n -k2 -r | head -n5,

                a).sort - for sorting  
                b).-n - numerically  
                c).-k2 - column position  
                d).-r - recursively  
                e).head - sort results from top  
                f).-n5 - top 5 results
